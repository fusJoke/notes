# 创建者模式

1. 单例模式singleton pattern
   一个类确保只有一个对象被创建

   1.1单例模式的实现
   	饿汉式：类加载就会导致对象的创建
   	饱汉式：类加载不会导致对象被创建，而是首次使用该对象	
   	

   ```java
   //饿汉式
   public class Singleton{
     private Singleton(){}
     private static Singleton instance = new Singleton();
     
     public static Singleton getInstance() {
       return instance
     }
   }
   ```

   ```java
   //饱汉氏
   //加同步锁
   public class Singleton{
     private Singleton(){}
     private static Singleton instance;
     
     public static synchronized Singleton getInstance() {
       if (instance == null) {
         instance = new Singleton();
       }
       return instance
     }
   }
   //双重检查锁
   public class Singleton{
     private Singleton(){}
     private static Singleton instance;
     
     public static Singleton getInstance() {
       if (instance == null) {
         synchronized(Singletion.class{
           	if (instance == null) {
               instance = new Singletion();
             }
         })
       }
       return instance
     }
   }
   
   //静态内部类
   public class Singleton{
     private Singeton() {}
     private static class SingetonHolder {
       private static final Singleton INSTANCE  = new Singeton();
     }
     public static Singleton getInstance(){
       return SingetonHolder.INSTANCE
     }
   }
   
   ```
