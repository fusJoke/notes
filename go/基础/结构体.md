什么是结构体？
	结构体是将零个或者多个任意类型的命令变量组合在一起的聚合数据类型。

```go
//结构体Employee
type Employee struct {
	ID				int
  Name			string
  Address		string
  Dob				string
  Postion 	string
  Salary		string
  ManagerId	int
}
//结构体变量
var dilbert Employee
```

结构体的成员变量通过点号(.)访问,或者获取成员变量的地址，通过指针来访问。点号也可以用在结构体指针

```go
dilbert.Salary -= -5000
position := &dilbert.Salary
*position = “Senior ” + *postion

var employeeOfTheMonth *Employee = &dilbert
employeeOfTheMonth.Position += " (proactive team player)" 
(*employeeOfTheMonth).Position += " (proactive team player)" 
//为什么提供通过结构体指针访问结构体变量的方式
func EmployeeById(id int) *Employee {}
id := dilbert.ID
EmployeeById(id).Salary = 0 //如果函数EmployeeById的返回值是指针地址，可以直接加.访问结构体
```

成员变量的顺序。成员变量的首字母大写表示可导，小写不可导

```go
type Employee1 struct {
	ID				int
  Name			string
}
type Employee2 struct {
  Name			string
	ID				int
}
//Employee1 和 Employee2 是不相同的结构体
```

命名结构类型s不可以定义拥有相同结构体类型的s的成员变量(禁止套娃)。但是可以定义s中可以定义一个s的指针类型

```go
type tree struct {
	value			  int
	left, right	*tree
}
```

结构体字面量

```go
//结构体类型的值可以通过结构体字面量来设置
type Point struct{X, Y int}
p := Point{1,2}
//指定部分成员变量的值
ainm := Point{X:1} //未设置的Y就是零值
//大型结构体通常使用结构体指针的方式直接传递给函数或者从函数中返回
//go是按值调用
```

结构体比较。结构体的所有都可以比较，那么这个结构体就是可比较的。使用==或者!=。  可比较的结构体类型可以作为map键的类型

```go
p := Point{1,2}
q := Point{2,1}
p.X == q.X //false
p==q //false

hits := make(map[address]int)
hits[address{"asdas2",443}]++
```
